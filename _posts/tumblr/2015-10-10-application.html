应用发展阶段&相关应用类型

  • PC:
    • Desktop Application： 原生的PC应用
  • 网络:
    • Distributed Application: C/S架构, P2P
    • Web Application（浏览器的普及）: B/S架构，避免安装和客户端更新，方便用户使用
  • 云计算:
    • Cloud Application（SaaS）: 部署在云端的应用，具有云的特性
  • 移动互联网:
    • Mobile Application: 原生的移动应用
    • Mobile Web Application: 针对移动互联网的Web应用
    • Mobile Cloud Application: 针对移动互联网的云应用
    • Hybird Application: Mobile App和Cloud App的混合，兼具两者的优点
    • Hybird Mobile App: 维护一份开发代码，减少因多平台带来的多平台开发和维护问题

其他相关概念

  • native application: 基于特定硬件平台编写的app。
  • emulated application： 在运行时转换成机器语言
  • web app: 基于浏览器开发的，被存储在服务器端。
  • true native： 相对于通过在运行时编译或解释的应用（Android App）来讲，预编译成机器语言的原生应用称为true native。
